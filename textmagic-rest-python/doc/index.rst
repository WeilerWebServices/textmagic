TextMagic Python Client API
===========================

`TextMagic <http://www.textmagic.com>`_ is an All-InOne Bulk SMS Service for Business.
The TextMagic Client API provides a convenient way
to access the `TextMagic Rest API <https://www.textmagic.com/docs/api/>`_ from Python.
